# nexus-platform-worker

TypeScript (Node.js) worker for Nexus. Handles content extraction, LLM distillation, embedding generation, taxonomy clustering, and AI-powered search.

## Tech Stack
- TypeScript + Node.js
- Redis Streams (ioredis)
- Claude API (Anthropic SDK)
- Cohere (embeddings)
- Firecrawl (web extraction)
